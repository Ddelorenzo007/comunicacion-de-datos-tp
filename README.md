## Descripción

Aplicación web desarrollada para la materia Comunicación de Datos.

La aplicación permite simular el proceso de digitalización de imágenes mediante técnicas de muestreo y cuantización, permitiendo comparar visualmente los resultados obtenidos a partir de diferentes configuraciones.

---

### Back-End

* Python 3.11+
* Flask
* Pillow
* NumPy

### Front-End

* CSS

* Visual Studio Code
* Git

---

## Funcionalidades

### Digitalización de Imágenes

* Carga de imágenes.
* Muestreo con distintas resoluciones.
* Reducción de profundidad de bits.
* Comparación entre imagen original y digitalizada.
* Cálculo de tamaño estimado.
* Visualización de estadísticas del procesamiento.

## Instalación y ejecución

### 1. Clonar el repositorio

```bash
git clone https://github.com/vicmoritan/comunicacion-de-datos-tp
```

### 2. Ingresar al proyecto

```bash
cd tp-comunicacion-de-datos
```

### 3. Crear entorno virtual

```bash
python -m venv venv
```

### 4. Activar entorno virtual

Windows:

```bash
venv\Scripts\activate
```

### 5. Instalar dependencias

```bash
pip install -r requeriements.txt
```

### 6. Ejecutar la aplicación

```bash
python app.py
```

### 7. Abrir en el navegador

```text
http://127.0.0.1:5000
```

## Instrumento de ejecución

La aplicación puede ser ejecutada localmente siguiendo los pasos indicados en la sección "Instalación y ejecución".

```bash

```

No requiere base de datos ni configuraciones adicionales.
